Cómo usar tu cuenta de desarrollador

Acceso a tus archivos en el directorio personal: /home/tu_nombre_de_usuario

Acceso mediante SAMBA

Para acceder a tu directorio personal a través de Samba, sigue estos pasos:

  1. Abre una ventana de explorador de archivos en tu computadora.
  2. En la barra de direcciones, escribe la siguiente ruta: smb://workstation.sixtema.int/tu_nombre_de_usuario.
  3. A continuación, deberías poder acceder a tu directorio personal.

Acceso mediante SSH

Para acceder a tu directorio personal a través de SSH, sigue estos pasos:

  1. Abre una terminal.
  2. Con el comando ssh tu_nombre_de_usuario@workstation.sixtema.int, inicia sesión en el sistema.
  3. Accede a tu directorio personal con el comando cd /home/tu_nombre_de_usuario.
  4. Ahora podrás ejecutar comandos, ejecutar las aplicaciones de tus proyectos.

Acceso mediante SFTP

Para acceder a tu directorio personal a través de SFTP, sigue estos pasos:

  1. Abre tu cliente SFTP favorito.
  2. Con los siguientes parametros, inicia sesión en el sistema.
  • Host: workstation.sixtema.int
  • Usuario: tu_nombre_de_usuario
  • Contraseña: tu_contraseña
  • Directorio remoto: /home/tu_nombre_de_usuario
  • Puerto: 22

Acceso a tu espacio público mediante navegador web.

Para acceder a tu espacio público a través de un navegador web, sigue estos pasos:

  1. Abre el navegador web.
  2. En la barra de direcciones, ingresa la dirección http://workstation.sixtema.int/site/tu_nombre_de_usuario.
  3. A continuación, deberías poder acceder a tu espacio público y explorar su contenido.

Acceso a la base de datos MariaDB

Para acceder a la base de datos MariaDB, tienes varias opciones:

Acceso mediante SSH

  1. Abre una terminal.
  2. Con el comando ssh tu_nombre_de_usuario@workstation.sixtema.int, inicia sesión en el sistema.
  3. Accede a la base de datos con el comando mysql -u tu_nombre_de_usuario -p.
  4. Ingresa con tu contraseña.
  5. Ahora podrás ejecutar comandos SQL.

Acceso mediante con gestor de base de datos

Para acceder , sigue estos pasos:

  1. Abre tu gestor de base de datos favorito.
  2. Utiliza una conexión que permita usar conexión SSH y conexión con MariaDB a la vez.
  3. Utiliza los parametros SSH siguientes:
    • Host: workstation.sixtema.int
    • Usuario: tu_nombre_de_usuario
    • Contraseña: tu_contraseña
    • Puerto: 22
  4. Utiliza los parametros de conexión con MariaDB siguientes:
    • Host: localhost
    • Usuario: tu_nombre_de_usuario
    • Contraseña: tu_contraseña
    • Base de datos: tu_nombre_de_usuario
    • Puerto: 3306

Utilizar docker compose

Para utilizar docker compose, sigue estos pasos:

  1. Abre una terminal.
  2. Con el comando ssh tu_nombre_de_usuario@workstation.sixtema.int, inicia sesión en el sistema.
  3. Situáte en el directorio de tu proyecto.
  4. Crea un archivo docker-compose.yml con el contenido de tu proyecto.
  5. Ejecuta el comando docker-compose up -d para iniciar los contenedores.
  6. Ejecuta el comando docker-compose down -v para detener los contenedores.
