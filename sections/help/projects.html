Crear proyectos con Entornos Virtuales

Proyecto en Python

  1. Crea una carpeta para tu proyecto: mkdir mi_proyecto_python
  2. Accede a la carpeta de tu proyecto: cd mi_proyecto_python
  3. Crea un entorno virtual de Python: python -m venv venv
  4. Activa el entorno virtual: source venv/bin/activate
  5. Instala las dependencias de tu proyecto en el entorno virtual, por ejemplo: pip install nombre_dependencia
  6. Desarrolla tu proyecto en Python dentro del entorno virtual.
  7. Desactiva el entorno virtual cuando hayas terminado: deactivate

Proyecto en Node.js

  1. Crea una carpeta para tu proyecto: mkdir mi_proyecto_node
  2. Accede a la carpeta de tu proyecto: cd mi_proyecto_node
  3. Inicializa un proyecto de Node.js: npm init -y
  4. Crea un archivo `.npmrc` en la raíz de tu proyecto con el siguiente contenido para establecer la ubicación del directorio de Node.js en tu proyecto: 
    
prefix=${PWD}/node_modules
  5. Instala las dependencias de tu proyecto localmente en el directorio `node_modules`: npm install nombre_dependencia

Proyecto en PHP

  1. Crea una carpeta para tu proyecto: mkdir mi_proyecto_php
  2. Accede a la carpeta de tu proyecto: cd mi_proyecto_php
  3. Crea un archivo `composer.json` para gestionar las dependencias de PHP en tu proyecto. Por ejemplo: 
    
{
"require": {
    "nombre_dependencia": "*"
}
}
  4. Instala las dependencias de tu proyecto en el directorio local: php composer install
